\begin{abstract}
Este trabajo se basa en el desarrollo de un sistema m\'inimo capaz de manejar video en dos dimensiones de una manera muy sencilla. Para facilitar el desarrollo se propone la creaci\'on de un ambiente de desarrollo flexible que incluya un simulador y una tarjeta.

El simulador permitir\'a crear programas y probarlos sin necesidad de tener un dispositivo de hardware fabricado. Para permitir extender el sistema, el simulador incluye un mecanismo por medio del cual es posible simular dispositivos externos.

Con esto se desarrollar\'a un sistema flexible y sencillo que permita dise\~nar hardware y software a la medidad.
\end{abstract}
